# Security Policy

## Supported Versions

As this is in early development the latest version will be supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :white_check_mark: |
| <0.1.0  | ✖️                 |

## Reporting a Vulnerability

The best way to report a security vulnerability in `dwgen` is to submit an Issue for the project. If you want to keep the disclosure private, please email security@digicratic.com
